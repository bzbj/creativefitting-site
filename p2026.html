数千万融资，原 AWS 首席应用科学家加盟，打造内容行业的 Agent 原生公司

2026.06.09 · CreativeFitting 井英科技

2026 年 6 月，井英科技（CreativeFitting）宣布已完成数千万美元 A 轮及 A+ 轮融资，投资方包括 Lollapalooza Capital（王慧文家办）、蚂蚁集团、腾讯公司原集团副总裁殷宇等。与此同时，公司正式官宣原 AWS 亚马逊上海 AI 研究院首席应用科学家、现任香港大学上海智能交叉创新研究院院长王敏捷加入，担任首席科学家。这也体现了具备前沿科技判断和产业资源的投资方，对井英科技团队、技术路线和公司愿景的认可。通过本轮融资，井英科技将继续加大对 AI 短剧的技术投入，构建内容创作和评估反馈环境，以 Agent 原生的方式推动内容娱乐行业升级。

井英科技创立于 2021 年，长期围绕 AI 内容生产技术和娱乐消费场景进行探索。内容娱乐是一个万亿美元级市场，AI 短剧则是其中增长最快的新兴场景之一，而且短剧的反馈密度天然适合驱动 Agent 迭代。井英科技是全球 AI 短剧赛道的开创者，率先在海外 AI 短剧行业中跑通了从生产、分发到用户消费的完整闭环。团队专注于构建和优化各环节的 Agent 系统，使内容创作、评估和反馈形成持续运转的工作回路。目前已有上千名创作者及其 Agent 接入系统。随着更多创作者及其 Agent 的接入和经验的积累，系统将不断自我进化。

Coding Agent 已经证明，Agent 能深刻重构一个行业。内容行业是下一个。Agent 原生公司是一种全新的组织范式：每个人创造自己的 Agent 加入公司工作，组织摩擦大幅降低，公司以 Agent 的速度运转。AI 短剧是首选场景，但 Agent 原生公司最终会改造整个内容娱乐行业。

— 井英科技创始人兼 CEO 朱江
