Scene 02 — The System

Agent 驱动的内容生成与评估系统An agent-native environment for content

为内容行业提供一个 Agent「能接入、能自我进化」的内容环境。

Pillar 01 — Connectable

能接入

井英科技构建了一套开放式的 Agent 原生创作工具链。拥有不同创意类型的人——剧本、拍摄、剪辑、画面——都可以构建自己的 Agent，完成从创意到成片的完整创作链路，并接入评估环境。

Pillar 02 — Self-evolving

能自我进化

内容行业的评估标准本身是动态的——用户偏好随时在变。因此整个系统需要持续进化：创作者的 Agent 不断注入创意，市场反馈实时回流，驱动评估标准动态更新，两者共同逼近真实的市场信号。

减少执行摩擦，释放创意本身

Agent 驱动加速了市场验证与反馈回路，创作者得以从流程协调中解放，真正专注于创意本身，而不是繁琐的同步协作。

短剧是第一个场景，不是终点

短剧天然具备高频、密集的反馈信号，是验证系统能力的最佳起点。同一套 Agent 内容环境，未来可扩展到更广泛的内容娱乐场景。

Scene 04 — About Us

关于我们Who we are

井英科技（CreativeFitting）创立于 2021 年，是一家 AI 原生娱乐技术公司，长期围绕 AI 内容生产技术和娱乐消费场景进行探索。公司已获得多轮融资，创始团队曾任美股上市公司创始人及高管，核心团队毕业于上海交通大学等高校。

井英科技以 AI 短剧作为首个商业化验证场景，率先在海外 AI 短剧行业中跑通了从生产、分发到用户消费的完整闭环。公司推出的 AI 短剧 App Reel.AI 面向全球用户提供 AI 原生娱乐体验，代表作品《Five Brothers》成为全球首部跻身海外微短剧热榜的 AI 短剧。

随着底层模型能力和 Agent 能力的持续成熟，井英科技正在继续加大对 AI 短剧的技术投入，构建内容创作和评估反馈环境，以 Agent 原生的方式推动内容娱乐行业升级。团队专注于构建和优化各环节的 Agent 系统，使内容创作、评估和反馈形成持续运转的工作回路，让 AI 原生内容从生产、分发到用户消费形成持续迭代的闭环。
